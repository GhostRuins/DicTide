# Changelog

All notable changes to DicTide are documented in this file.

## [0.2.0] - 2026-04-17

### Added
- Configurable global hotkey with next-key capture.
- Toggle and hold hotkey activation modes.
- Expanded faster-whisper model presets plus custom model path/HF ID input.
- Single-file installer build path via Inno Setup.

### Changed
- Rebranded project/runtime/packaging names from WhisperDictation to DicTide.
- Added first-run migration from legacy `%LOCALAPPDATA%\WhisperDictation` to `%LOCALAPPDATA%\DicTide`.
